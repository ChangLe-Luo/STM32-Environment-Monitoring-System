# STM32环境监测系统

基于STM32F10x系列微控制器的多参数环境监测与响应系统。

## 项目概述

本项目实现了一个综合环境监测系统，能够实时采集空气质量、光照强度、温湿度等环境参数，并根据预设阈值自动控制执行器（电机、舵机、蜂鸣器等）进行环境调节。

## 硬件组成

### 主控芯片
- STM32F10x系列微控制器

### 传感器模块
- **MQ135** - 空气质量检测（模拟/数字模式可选）
- **光敏传感器** - 光照强度检测（模拟/数字模式可选）  
- **DHT22** - 温湿度传感器

### 执行器模块
- **直流电机** - 用于通风控制（PWM调速，正反转）
- **舵机** - 角度控制（0-180°）
- **蜂鸣器** - 声音报警
- **LED灯** - 光强指示（PWM调光）

### 显示模块
- **OLED显示屏**（I2C接口） - 实时数据显示

## 主要功能

### 数据采集
- 实时监测空气质量（PPM值）
- 采集光照强度（0-3500范围）
- 读取温湿度数据（带校验）

### 智能响应
```c
// 空气质量分级响应
1级(>30ppm): 电机全速反转 + 蜂鸣器报警
2级(20-30ppm): 电机半速反转 + 蜂鸣器提醒
3-5级: 电机停止 + 舵机归零

// 光照自适应
LED亮度根据光照强度自动调节
```

### 数据显示
OLED显示屏实时显示：
- 空气质量等级（VeryBad - VeryGood）
- 光照等级（VeryDark - VeryBright）
- 温度值（xx.xC）
- 湿度值（xx.x%）

## 使用说明

1. **初始化系统**
```c
EMS_Init(); // 初始化所有外设
```

2. **主循环处理**
```c
while(1) {
    EMS_Update();     // 更新传感器数据
    EMS_Response();   // 执行控制逻辑
    EMS_Display();    // 刷新显示
    Delay_ms(2000);   // 2秒周期
}
```

## 项目特点

- **模块化设计** - 各功能模块独立，便于维护和扩展
- **自适应控制** - 根据环境参数自动调整执行器状态
- **实时显示** - OLED提供直观的数据反馈
- **多级报警** - 分级响应机制，适应不同环境状况
